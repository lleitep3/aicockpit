# AGENTS.md (Global Rules)
The following are user-defined rules that you MUST ALWAYS FOLLOW WITHOUT ANY EXCEPTION:

1. **Aesthetics are critical**: When generating UI code, avoid generic colors. Use modern typography, glassmorphism, and subtle micro-animations.
2. **File Links**: You MUST create clickable links for all files and code symbols using github style markdown links with the `file://` scheme (e.g., `[filename](file:///path/to/file)`).
3. **No Placeholders**: Never generate placeholder code. If an image is needed, use the `generate_image` tool to create a real one.
