# Goose Workflows (Recipes)

No ecossistema do Goose, os Workflows recebem o nome nativo de **Recipes** (Receitas). Elas são estruturas projetadas especificamente para compartilhar e reutilizar fluxos de trabalho guiados por IA.

## Como Funcionam
As *Recipes* são templates instrucionais que empacotam conhecimento operacional. Em vez do desenvolvedor ter que digitar todo um procedimento manual complexo a cada vez, ele apenas invoca a receita.

- **Automação de Tarefas:** Permitem codificar tutoriais, guias de refatoração ou checklists complexos.
- **Engenharia de Prompt:** As receitas funcionam injetando um conjunto estruturado de prompts passo a passo no contexto do agente.

## Como o AICockpit Mapeia
O AICockpit mapeia a entidade "Workflow" do usuário diretamente para o formato de *Recipes* suportado nativamente pelo Goose. O compilador do adaptador traduz a estrutura agnóstica do Cockpit para um arquivo pronto para ser lido pelo motor do Goose.
