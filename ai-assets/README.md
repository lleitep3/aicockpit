# AI Assets - Library and Extensions
This directory stores the library of assets that expand the capabilities of the AICockpit. 
Unlike the internal engine logic, these are modular resources that define how AI agents behave and what tools they have access to.

## Subdirectories:
- **knowledge-base/**: Contains guides, documentation, and external knowledge that the KB engine indexes for AI agents to query.
- **skills/**: A collection of reusable AI "skills". These modular units of logic provide agents with specific capabilities (e.g., package creation, code generation, testing).
- **packages/**: (Planned) Will contain definitions and manifests for modular packages that can be installed into the cockpit ecosystem.
